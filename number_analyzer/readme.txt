# 📊 Number Analyzer

## 📌 Описание
Это простой Python-проект для анализа чисел, введённых пользователем.  
Программа выполняет базовую статистику и разделяет числа по категориям.

---

## ⚙️ Функционал
- Ввод произвольного количества чисел
- Вычисление суммы
- Поиск максимального и минимального значения
- Подсчёт положительных чисел
- Подсчёт отрицательных чисел
- Подсчёт нулей

---

## 🚀 Как запустить
1. Установите Python 3
2. Откройте терминал в папке проекта
3. Запустите файл:main.py

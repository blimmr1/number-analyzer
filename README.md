# number-analyzer
This is a simple Python project for analyzing user-entered numbers. The program performs basic statistics and categorizes numbers.
⚙️ Functionality
- Entering any number of numbers
- Calculating a sum
- Finding the maximum and minimum values
- Counting positive numbers
- Counting negative numbers
- Counting zeros

---

 🚀 How to run
1. Install Python 3
2. Open a terminal in the project folder
3. Run the file:main.py
